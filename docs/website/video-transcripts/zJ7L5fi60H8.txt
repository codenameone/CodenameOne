we continue the trend of simple services
with the media service which is almost
as simple as the notification service

repository only but it needs to
authenticate some actions against the
user repository

all the values from the client with the
exception of time which we store in the
server to prevent abuse
we need an auth token to make sure the
media is saved under the correct user

so it can be referenced by the user

any authentication

isn't spoofing a request
