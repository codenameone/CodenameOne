About Us Form
the about form of the application is
probably the simplest form we have
it just displays the placeholder html
i'm not a big fan of html
based uis but one thing html does
relatively well is about forms and it's
really easy to just move this out
notice i'm using set url hierarchy which
allows us to package the entire html
source tree with css javascript etc
under the html package this will display
implicitly in a portable way for the
devices
Placeholder
