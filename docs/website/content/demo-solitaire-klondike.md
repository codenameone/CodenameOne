## SOLITAIRE KLONDIKE

A fully featured Solitaire Klondike game with multiple levels of undo

![](/demos/solitaire-klondike-slide-1-3.png)

![](/demos/solitaire-klondike-slide-2-3-1.png)

The Solitaire Klondike app was adapted from the old Poker demo resources so we will be able to submit the app to the stores and provide a fully installable demo that will more accurately represent the way things look when you build the native app.

Notice that the JavaScript HTML5 demo is not what we have on the device where applications are translated to native code!  
Since Apple doesn’t allow demos on itunes and doesn’t allow installing apps without itunes we can’t distribute the native app demos, you would need to build from source with an itunes demo account.
