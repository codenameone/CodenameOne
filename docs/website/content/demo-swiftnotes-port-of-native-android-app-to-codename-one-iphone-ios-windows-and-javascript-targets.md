This demo is a part of a tutorial on converting Android native apps to Codename One apps thus allowing them to run on iOS, Windows & even the web.  
The UI and UX of this demo is derived from the native Android app, we tried to keep it as close as reasonably possible to the original to avoid confusion. You can check out the original native Swiftnotes app [here](https://play.google.com/store/apps/details?id=com.moonpi.swiftnotes) and its source code [here](https://github.com/adrianchifor/Swiftnotes). Our versions and sources are listed below.

Notice that the JavaScript HTML5 demo is not what we have on the device where applications are translated to native code!
