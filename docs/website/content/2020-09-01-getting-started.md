## Install the plugin using the Plugins Center

Click File-> Settings-> Plugins and use the search field to find and install the Codename One plugin.

NOTE: The plugin requires IntelliJ/IDEA, the plugin does not work on Android Studio.

## Install the plugin using the plugin manager

Click the Tools -> Plugins menu option and search for Codename One in the available plugins section. Install it.

NOTE: The plugins.netbeans.org server has been down frequently in the past couple of months preventing automatic installation. Please follow the instructions [here](https://www.codenameone.com/blog/netbeans-plugin-update-center.html) as a workaround if the instructions above don't work.
