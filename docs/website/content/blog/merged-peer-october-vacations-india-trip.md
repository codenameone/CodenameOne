the `android.newPeer` flag as all non-versioned requests will use the new peers. This is a good thing as it will allow  
us to focus on one implementation moving forward.

We would like to bring the new peer implementations to other platforms but so far this has proven itself somewhat  
challenging especially with our resource constraints. Ideally we wanted to have this for 3.6 but itâ€™s possible that it
