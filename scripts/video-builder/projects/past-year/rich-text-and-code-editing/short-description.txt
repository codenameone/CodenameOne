Two new cross-platform visual editors ship in PR #5272. RichTextArea is a WYSIWYG HTML editor, and CodeEditor is an IDE-style editor with syntax highlighting for eight languages, async code completion and diagnostics. Both run on a shared AbstractEditorComponent that talks to two interchangeable backends.
