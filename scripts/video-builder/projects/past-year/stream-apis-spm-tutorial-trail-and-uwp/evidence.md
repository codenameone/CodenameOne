## Thesis

Modernizing a Java mobile stack with streams and Swift Package Manager while retiring UWP

## Supported beats

- **Built-in Support for Java Stream APIs:** We have supported Java 8 language features for a long time, but streams remained one of those APIs that developers would miss the moment they tried to bring modern Java habits or shared code into a Codename One project. That changes now.
- **Swift Package Manager Support:** The other big update is that we added Swift Package Manager support to Codename One, and documented it in the Working with iOS section of the developer guide.
- **Tutorial Trail Refresh:** Over March 15-17, 2026 we refreshed a large portion of the tutorial trail and revised the written text across the course material.
- **UWP Removal from the Maven Plugin:** This is really an acknowledgment of reality more than a sudden functional change.
