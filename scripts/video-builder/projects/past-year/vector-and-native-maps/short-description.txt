A new maps API (PR #5264) brings mapping back into Codename One core with two components on one MapSurface API — a pure-vector MapView drawn entirely through Graphics, and a NativeMap whose provider (Apple MapKit, Google Maps) is chosen by a build hint and injected at build time.
