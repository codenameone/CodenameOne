## Thesis

What Java 17 syntax works in Codename One and where the experimental boundary remains

## Supported beats

- **How This Works:** You can use these new Java 17 projects with practically any JDK for the build itself. We tested this with JDK 21 and even JDK 25.
- **What Java 17 Syntax Works?:** At the moment this includes features like var, switch expressions and text blocks. Record support is planned for Codename One 7.0.229, slated for release on March 20, 2026.
- **What About Java 21 and 25?:** We do plan to add support for newer language levels in a coming update. That is a different challenge, because Android only supports up to Java 17 today.
- **Other Updates Worth Mentioning:** The Initializr now includes more advanced CSS theme editing with live preview, and there were several follow-up fixes to make theming behave more reliably.
- **Try Java 17 on a Real Project:** If you have been waiting for a more modern Java syntax level in Codename One, this is the time to give it a try.
