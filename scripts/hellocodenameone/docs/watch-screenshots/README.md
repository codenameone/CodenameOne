# Watch form-factor demonstration screenshots

Generated by running `hellocodenameone` in the Codename One JavaSE simulator with
the Apple Watch 45mm skin (`CN.isWatch()` == true), 396×484.

`AbstractGraphicsScreenshotTest` detects the watch form factor and emits **four
separate full-screen captures** — one per rendering variant (direct vs
mutable-image painting × anti-aliasing off/on) — instead of the single 2×2 grid
used on phone/tablet/desktop. The `*-PHONE-grid.png` files (if present) are the
phone baselines for contrast.

These are demonstration artifacts for PR #5251 and can be dropped before merge.
